# Bitwarden CLI

### [HOME](https://bitwarden.com) | [SOURCE](https://github.com/bitwarden/clients/releases/tag/cli-v2023.9.1)

Open source password management solution for individuals, teams, and business organizations - The CLI only variant

Bitwarden is described as 'the easiest and safest way to store your logins and passwords across all of your devices (iOS, Android, Chrome, Firefox, Opera, Windows, Mac, Linux, and more). Bitwarden is FREE and open source' and is a leading password manager in the security & privacy category.

#### Install:

```
kcp -i bitwarden-cli
```
#### To login:

```
bitwarden-cli login
```

#### NOTE:

Use `bitwarden-cli` instead of `bw` everywhere . . .
